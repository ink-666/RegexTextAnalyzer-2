## Функциональность

### 1. Регулярные выражения
- Подсчёт букв, цифр и слов в тексте
- Поиск строк, начинающихся с определённого слова
- Поиск строк, оканчивающихся заданным символом
- Замена частей текста по шаблону:
  - Удаление лишних пробелов
  - Замена цифр на символ `#`
  - Цензура слов (замена "тупогуб" на "****")

### 2. Указатели (unsafe code)
- Получение адресов переменных в памяти
- Изменение значений через указатели
- Работа с указателем на указатель (`int**`)
- Арифметика указателей при работе с массивами
- Фиксация массивов в памяти (`fixed`)
